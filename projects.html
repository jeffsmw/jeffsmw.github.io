Projects

Pic OutNovember-December 2016

An application for foodies: those who like to eat and love to take pictures of what they eat. This application brings up Instagram photos near a location and allows users to view and get directions to restaurants.

The project made with Ruby on Rails and postgreSQL, with a heavy emphasis on Javascript and Ajax. APIs in this project are Zomato, Instagram, and Google Maps. The project uses a scraping method to retrieve photos, and out of respect for Instagram's policies the app is not live.

Pet FinderNovember 2016

A Web application for pet enthusiasts to share information on lost, seen, and found pets. This weekend project was in collaboration with Envisioning Labs and had 8 members, and relied heavily on Git and version control.

I was responsible for creating the client and Ajax requests, however the client was not implemented as my role changed to designing the application interface according to the provided wireframes. Additionally I worked on making it mobile-friendly and also tested the application, verifying features worked as intended.

Solder Reflow Oven ControllerFebruary 2014

In a group of 6, we created a solder reflow controller using our knowledge in circuitry and 8051 Assembly language. The controller was implemented through an Altera DE2 board, a standard off-the-shelf toaster oven, and through a solid state relay box.

I worked on the software portion of this project, doing the majority of the coding. My tasks involved:

  • creating a state machine
  • using interrupts and clock cycles to run processes in parallel
  • implementing the LCD display and creating characters on the display
  • error checking and revision control using Git
