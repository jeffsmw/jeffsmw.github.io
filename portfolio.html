Solder Reflow Oven Controller

In a group of 6, we created a solder reflow controller using our knowledge in circuitry and 8051 Assembly language. The controller was implemented through an Altera DE2 board, a standard off-the-shelf toaster oven, and through a solid state relay box.

I worked on the software portion of this project, doing the majority of the coding. My tasks involved:
  • creating a state machine
  • using interrupts and clock cycles to run processes in parallel
  • implementing the LCD display and creating characters on the display
  • error checking and revision control using Git
